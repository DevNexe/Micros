# Redmi Mic - Android Audio Server (Direct Stream)

Это приложение превращает ваш телефон в полноценный HTTP-сервер аудио (аналог IP Webcam). Оно транслирует звук с микрофона напрямую по запросу.

## Как это работает
Приложение запускает HTTP-сервер на порту `8080`. Как только ваш Python-скрипт обращается к `http://IP_ТЕЛЕФОНА:8080/audio.wav`, приложение начинает захват звука и передает его в поток.

## Сборка через GitHub (для слабых ПК)
1. Создайте репозиторий на GitHub.
2. Загрузите все файлы из папки `RedmiMicApp`.
3. Перейдите во вкладку **Actions** -> **Build APK**.
4. Скачайте готовый APK из раздела **Artifacts**.

## Использование
1. Установите APK и дайте разрешение на микрофон.
2. На экране телефона появится адрес (например, `http://192.168.43.1:8080/audio.wav`).
3. Вставьте этот адрес в ваш Python-скрипт в переменную `url`.
4. Запустите Python-скрипт.

**Никаких промежуточных Flask-серверов больше не требуется!**
